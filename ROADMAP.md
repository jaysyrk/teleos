# Teleos Roadmap

## Vision
Enable explainable, language-agnostic rule engines for business logic, accessible from any programming language.

## Near-Term Goals
- [ ] Expand .teleos syntax (actions, assignments, alerts)
- [ ] Improve error messages and proof explanations
- [ ] Add more CLI commands (lint, format, export)
- [ ] Python: type hints and better exceptions
- [ ] VS Code extension for syntax highlighting

## Mid-Term Goals
- [ ] Native bindings for Go, Java, C#, C++
- [ ] WebAssembly build for browser/edge
- [ ] Rule versioning and hot-reload
- [ ] Performance optimizations for large rule sets

## Long-Term Goals
- [ ] Visual rule editor (web UI)
- [ ] Natural language to .teleos transpiler
- [ ] Integration with workflow/orchestration tools
